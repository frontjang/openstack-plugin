OpenBaton Openstack Plugin
----------------

OpenBaton is an open source project providing a reference implementation of the NFVO and VNFM based on the [ETSI][NFV MANO] specification, is implemented in java using the [spring.io] framework. It consists of two main components: a NFVO and a generic VNFM. This project **openstack-plugin** contains an implementation of a plugin for OpenBaton system.

##### How-To

Please see the [Plugin SDK][plugin-sdk-link]

##### Installation

No installation required.
